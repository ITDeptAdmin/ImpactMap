ℹ️ How It Works

1 Staff upload an updated Master CSV to GitHub.
2 GitHub Actions builds the map automatically.
3 Safe coordinates are filled in automatically. Uncertain ones appear in the Review Queue below.
4 A Pull Request is opened with suggested coordinates ready to apply.
5 A user with write access reviews and merges the PR to approve.
